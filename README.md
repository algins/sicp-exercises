# sicp-exercises
Solutions of the SICP (Structure and Interpretation of Computer Programs) exercises

### Requirements:
* Racket
* Git
* Make

### Setup:
```sh
$ git clone https://github.com/algins/sicp-exercises.git
$ cd sicp-exercises
$ make install
```

### Run tests:
```sh
$ make test
```

### Run linter:
```sh
$ make lint
```
